# 11. With variable substitutions by name (hint: by using dict)
# dict comprehension
dict = {"apples": 5, "peaches": 7}
print(f"Anna has {apples} apples and {peaches} peaches.")

цей код не працює: зсилатися на словник краще через dict['apples'], або dict.get('apples'), 
якщо хочеш доступитися до значення ключа.

або ще такий метод (знайшов у інших домашніх завданнях):
dict = {'apple' : 3, 'peache' : 2}
print("Anna has {apple} apples and {peache} peaches.".format(**dict))
